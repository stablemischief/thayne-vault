## IMPLEMENTATION NOTES

### Intro Audio Handling
- Every product page lists tracks separately (Introduction / Meditation)
- **Never include Introduction time in user-facing "meditation duration"**
- For products with "Combined Breath + Meditation" track, offer user the choice: "With breath technique (Xmin) or Meditation only (Ymin)?"
- For products where guided portion ends and music continues (Restorative Sleep, Generating Abundance), show the guided time + note "music continues for integration"

### Source & Availability
- All meditations are exclusively on drjoedispenza.com unless noted
- No verified free YouTube links — community strongly advises against unofficial uploads
- Walk for the World = $0 / free

### Personal Library Feature (Phase 5)
- Add toggle per meditation: "I own this" → flag in user profile
- Recommendation engine filters to owned first, then suggests purchase options for others

### Copyright
- Never reproduce audio or transcription content
