# Deck Build Instructions

## Tool: Canva (recommended starting point)
- Go to canva.com → "Presentations" → start from a clean template
- Apply Stable Mischief brand colors (see brand-identity.md skill)
- 16:9 widescreen format
- Export as PDF when complete

## Reference: Agrellus Deck
- Structure: 14–15 slides, clean layout, one key message per slide
- Visual style: dark accent + white, logo prominently on cover
- Each product gets its own slide with TAM/SAM callout

## File naming when complete:
- `StableMischief-Investment-Deck-v1.pdf`
- `StableMischief-Executive-Summary-v1.pdf`
